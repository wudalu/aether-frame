# Summary Rewrite

When user asks to summarize or rewrite long text:
1. Preserve factual meaning and constraints.
2. Remove repetition and non-essential filler.
3. Return concise output in the style requested by the user.
4. If user input is ambiguous, ask one clarifying question before rewriting.
